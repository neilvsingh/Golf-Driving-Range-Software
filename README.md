### Description
The main problem I have when hitting golf balls at a driving range is running out of balls. 
With my design, we golfers will be able to notify golf course employees when their balls run dry, 
to be refilled promptly and improve the efficiency of the course

### Project Resources
- Hardware used includes two Raspberry Pi Pico W boards. Pico W used as they include RF transmitters/recievers
- Programmed Pico W using C++ to control hardware functionality and enable reliable wireless signal communication
